＊＊＊　ゲームタイトル　＊＊＊

シューティングパックマン
(c)2017-2018, 1班

＊＊＊　ストーリー　＊＊＊

宇宙人から逃げて食料（マフィン）を手に入れるシューティングゲームです。
銃で攻撃し宇宙人の動きを止めて食料（マフィン）をたくさん集めて高得点を狙いましょう。


＊＊＊　遊び方　＊＊＊

＝＝　操作　＝＝
↑↓←→　移動
spaceキー　弾を発射

＝＝　仕様　＝＝
時間100秒以内にプレイヤーを操作し食料（マフィン）を集めます。
敵は移動しながら攻撃してきます。敵や攻撃された弾に当たるとゲームオーバーになります。
プレイヤーは攻撃することができ、弾に当たった敵は動きを止めます。
時間内に敵や敵の弾に当たらならければゲームクリアです。取ったマフィンの合計（スコア）が表示されます。
